<h1 align="center">Ecoleta</h1>

<h3>♻️ Sobre</h3>
<p>O Ecoleta é um marketplace de coleta de resíduos. Com ele, é possível ajudar as pessoas a encontrarem pontos de coleta de resíduos de forma eficiente.</p>

# Instalar dependências
<ul>
    <li>É necessário ter o <a href="https://nodejs.org/en/">NodeJS</a> instalado.</li>
    <li>Também é necessário ter o <a href="https://www.npmjs.com/">NPM</a> ou 
        <a href="https://yarnpkg.com/">Yarn</a>.
    </li>
    <li>É preciso ter o <a href="https://expo.io/">Expo</a> instalado de forma global.</li>
</ul>

## API
<hr>

<pre>
    yarn install
    <span>#Iniciar banco de dados</span>
    yarn knex:migrate
    yarn knex:seed
    yarn dev
</pre>

## Front-End
<hr>
